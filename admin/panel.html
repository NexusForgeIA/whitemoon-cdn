Alta de nuevo cliente

1 Generar token: formato WM-SECTOR-NNN (ej: WM-DENTAL-001)
2 Añadir a licenses.json:
{
  "licenses": {
    "WM-DENTAL-001": {
      "biz": "Nombre del Negocio",
      "domain": "dominiocliente.es",
      "pack": "advance",
      "active": true,
      "created": "2026-04-15",
      "expires": "2027-04-15",
      "contact": "email@cliente.com",
      "phone": "912345678",
      "notes": "Pack Advance 99€/mes"
    }
  }
}
3 Subir a GitHub: git add . && git commit -m "Alta: NombreCliente" && git push
4 Pegar en la web del cliente (antes de </body>):
<script src="https://TUUSUARIO.github.io/whitemoon-cdn/chat.js"
  data-token="WM-DENTAL-001"
  data-color="#4A90D9"
  data-services="Servicio 1,Servicio 2,Pedir cita"
  data-bot-name="Lucía"></script>

Desactivar cliente (impago)

1 Abrir licenses.json
2 Cambiar "active": true → "active": false
3 git commit -am "Desactivar: NombreCliente" && git push
El chatbot desaparece de su web en minutos. El cliente no ve error — solo desaparece el botón.

Reactivar cliente

Cambiar "active": false → "active": true y hacer push.

Seguridad

Sin token → no carga nada
Token inactivo → no carga nada
Dominio incorrecto → si copian el código en otra web, no funciona
Licencia expirada → se desactiva automáticamente
Si un informático copia el script → el token está vinculado al dominio. Si dejan de pagar → desactivas el token y el chatbot desaparece de su web.
